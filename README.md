# GoogleDriveClient    
After authorization via Google, the application receives files that are located on Google Drive.When you click on a cell, you can share the file.    


## Stack and technologies    
- Swift
- UIKit
- GoogleSignIn, GoogleAPIClientForREST
- Kingfisher
